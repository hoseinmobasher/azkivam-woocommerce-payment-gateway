This plugin adds Azkivam credit payment gateway to Woocommerce.

== Description ==
This plugin is used to add [Azkivam](https://azkivam.com) credit payment gateway to your WooCommerece based shopping website.
Right after installing the plugin, you should setup your payment with appropriate apiKey, merchantId and paymentUri that you get from Azkivam.
